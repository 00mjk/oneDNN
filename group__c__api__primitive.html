mkldnn_primitive_at_t MKLDNN_API mkldnn_primitive_at ( const_mkldnn_primitive_t  primitive,
size_t  output_index 
)

Creates a dnn_primitive_at structure from a primitive and output_index.

This function only fills in the data structure and does not check whether parameters are correct. The actual error checking is done when the resulting dnn_primitive_at structure is passed to a primitive creation function.

Creates a memory primitive with a memory_primitive_desc and data_ptr.

If data_ptr is NULL, memory is initialized with internally allocated memory. primitive_destroy() function takes care of the allocated memory.

size_t MKLDNN_API mkldnn_memory_desc_get_number_of_elements ( const mkldnn_memory_desc_t *  memory_desc,
int  with_padding 
)

Returns the number of elements in memory_desc memory descriptor.

If with_padding is zero, the function returns the number of elements not including the padding, otherwise, the function returns the number of elements including the padding. If memory_desc describes a NULL tensor, format is not fully specified (for example: memory_desc->format == any), or memory_desc is inconsistent, the function returns zero.

size_t MKLDNN_API mkldnn_memory_desc_get_size ( const mkldnn_memory_desc_t *  memory_desc)

Returns the size (in bytes) that is required for the memory described by memory_desc.

If memory_desc describes a NULL tensor, format is not fully specified (for example: memory_desc->format == any), or memory_desc is inconsistent, the function returns zero.
